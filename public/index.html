How to Use

Slash Command

The syntax is /tl [lang] [text] Where [lang] is equal to a language code such as de, vi, ja, es. The source text doesn't have to be English. 

Message Action

Use Translate action from the actions menu for any Slack messages                                                              

Dialog

Enter /tl and the app shows a dialog with fields to select target language and enter your text.

Privacy Policy

Information Collected

I do not collect email addresses, usernames, or any other personal information. Nor do I collect the authentication token used to install the application.

Use of Information

When you use the /tl [lang] [text] slash command, I store the [lang] code and the word count of [text] used in order to know which languages are the most used.

I do not collect, store or care which words you translate.

Your Privacy Using /tl

All translations posted by /tl are visible by all channel members the command is issued from. If you wish to use it privately, you need to use it in a private channel, or in the Slackbot channel.
